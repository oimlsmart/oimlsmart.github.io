## Reading order

If you're new to the formal specs:

1. Read **01 (System Architecture)** for the overview.
2. Read **02 (Requirement & Conformance Model)** for the core entity types.
3. Skim **06 (YAML Schema)** and **07–09 (Calculation, Expressions, Table primitives)** for the declaration language.
4. Read **10 (Certification Workflow)** if you're integrating with the OIML-CS pipeline.
5. Read **11 (Variable & Symbol Model)** if you're extending the calculation engine.

For the narrative version of these specs — easier to read, less formal — see the [Developer Guides](/docs/guides/getting-started.html).
